# Avaliacao-da-evolucao-de-carreira-e-da-eficiencia-ofensiva-de-Yuri-Alberto
Projeto de análise de dados focado na avaliação da evolução de carreira e da eficiência ofensiva de Yuri Alberto, utilizando dados de performance no futebol, consultas em SQL e análise exploratória para apoiar decisões de recrutamento.
