Yuri Alberto Analises

1. Contexto do Projeto

Este projeto foi concebido com o objetivo de aplicar análise de dados esportivos a um cenário real de tomada de decisão no futebol, especificamente no contexto de avaliação de jogadores para recrutamento.

O jogador escolhido foi Yuri Alberto, atacante brasileiro, por apresentar uma trajetória com diferentes clubes e fases distintas de carreira, permitindo uma análise consistente de evolução e eficiência ofensiva.

2. Objetivo da Análise

O principal objetivo da análise é avaliar:

a evolução de performance do jogador ao longo das temporadas

sua eficiência ofensiva como atacante

se o perfil atual do jogador indica crescimento contínuo ou possível pico de performance

Os resultados simulam o tipo de análise utilizada por departamentos de scouting e análise de desempenho.

3. Perguntas-Chave

As seguintes perguntas orientam toda a análise:

A eficiência ofensiva do jogador evoluiu ao longo das temporadas com o ganho de experiência?

Quantas finalizações são necessárias, em média, para que o jogador marque um gol, e essa relação melhorou com o tempo?

Com base nos dados de performance, o jogador apresenta um perfil atrativo para decisões de contratação?

Essas perguntas foram definidas antes da coleta dos dados, garantindo uma análise guiada por hipóteses e não apenas por exploração aleatória.

4. Dados Necessários (Planejamento)

Para responder às perguntas propostas, serão necessários dados relacionados a:

temporadas

clubes

minutos jogados

gols

finalizações

jogos disputados (titular / reserva, se disponível)

Os dados deverão ser obtidos de bases públicas confiáveis e posteriormente tratados para análise.

5. Metodologia Prevista

A análise será conduzida utilizando:

SQL para consultas, agregações e criação de métricas

análise exploratória para identificação de padrões

métricas normalizadas (ex: gols por 90 minutos)

Os resultados serão apresentados por meio de gráficos simples e interpretações objetivas.

6. Escopo e Limitações

Este projeto não tem como objetivo:

avaliar impacto tático coletivo

analisar títulos ou desempenho do time

aprofundar em lesões ou cartões (fase futura)

O foco está exclusivamente na eficiência ofensiva individual.
